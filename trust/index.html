Trust and data policy

Local-first AI security software, with public claims kept source-of-truth clean.

Permanently Mobile builds self-hosted AI security tools. PM AIDefense is a local AI-agent defense scanner that checks repositories, prompts, dependencies, MCP configs, and agent runtimes without uploading code or spending LLM tokens. PM Privacy Guard classifies, sanitizes, tokenizes, and routes sensitive AI messages before they leave a user-controlled system.

Fast facts

  • Company: Permanently Mobile LLC.
  • Ownership model: one-time licenses for local-first tools.
  • PM AIDefense: local repository scanning, no code upload, no LLM token spend for inspection.
  • PM Privacy Guard: classifies, sanitizes, tokenizes, and routes sensitive prompts without storing message content.
  • Public PM AID package truth: @permanentlymobile/pm-aid@1.1.0, 74 shipped modules.
