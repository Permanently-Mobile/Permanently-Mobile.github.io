FAQ

Questions

What makes this different from Burp Suite? ▼
Burp Suite is a web application scanner. PM AI Defense was built from scratch for AI agent systems. It has 10+ modules that test prompt injection, memory poisoning, bridge exploitation, voice pipeline attacks, and other vectors that Burp has no concept of. It also includes full network recon, OSINT, and WiFi/Bluetooth testing that Burp doesn't cover. Different tools for different attack surfaces.
Does it phone home or require internet access? ▼
No. License validation is fully cryptographic and local. There is no activation server, no telemetry, no usage tracking. You can run it in air-gapped environments. The only time it needs network access is when scanning external targets or fetching vulnerability databases (Nuclei templates, npm advisories).
What's the difference between Base and Pro? ▼
Both tiers include all 47 scanner modules. Base restricts target scope to localhost and private IP ranges (127.x, 10.x, 172.16-31.x, 192.168.x) -- designed for hardening your own systems. Pro unlocks external target scanning, client-ready proof-mode reports, a custom module SDK for writing your own scanners, and a VPS deployment guide.
What do I need to run it? ▼
Node.js 20 or higher. That's it. No Docker, no external databases, no cloud accounts. Some modules optionally use tools like Nmap, Nuclei, or Semgrep if installed, but the core AI attack surface modules run standalone.
Can I use this to pentest client systems? ▼
Yes, with the Pro tier. Pro unlocks external target scanning and proof-mode reports with unredacted evidence, verification tokens, and timestamps suitable for client deliverables. Always ensure you have written authorization before scanning any system you don't own.
How often are modules updated? ▼
Module updates are included for one year from purchase. New attack vectors, updated vulnerability signatures, and module improvements are shipped as standard npm package updates. After year one, the software continues to work indefinitely -- you just stop receiving new module updates.
Is there a monthly fee? ▼
No. One-time payment. You own the software. No subscriptions, no per-scan fees, no usage limits. Compare that to Burp Suite Pro at $475/year or Invicti at $7,000/year.
