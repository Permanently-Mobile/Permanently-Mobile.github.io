Modules
Six attack classes. One scanner.
74 modules across the surfaces an AI agent actually gets hit on.
🔗
Prompt Injection
Instruction overrides, jailbreaks, and hidden directives buried in prompts, system files, tool output, and fetched content. Context-exploit detection across the agent loop.
📦
Supply Chain / Typosquats
Typosquatted and lookalike packages, suspicious install scripts, and dependency manifests that pull in poisoned or impersonated libraries.
🔑
Secret Exposure
API keys, tokens, private keys, and credentials committed to code, config, .env files, and prompts. Catches leaks before they ship.
🔧
MCP / Tool Abuse
Malicious or misconfigured MCP servers, over-broad tool permissions, API abuse, and tool-output poisoning that turns a helper into an attack vector.
🛡
Refusal-Bait Evasion
Malware seeded with weapons, CBRN, or jailbreak text to trip an AI scanner's safety refusal. PM AID detects the bait, quarantines it, and reads the real mechanics. A scanner that can't be gagged.
👑
Vault / Privilege Escalation
Unauthorized vault and secret-store access, privilege escalation paths, and runtime attempts to reach data or capabilities the agent was never granted.

FAQ
Common questions.
What does PM AIDefense scan? ▼
Codebases, dependencies, prompt files, MCP configs, agent tools, local dashboards, runtime settings, and repo metadata. The scanner looks for prompt injection, tool-output poisoning, exposed secrets, supply-chain typosquats, unsafe MCP permissions, localhost exposure, OSINT leaks, refusal-bait payloads, and other AI-agent attack paths.
What is the difference between Basic, Full, and Full Pro? ▼
Basic unlocks full findings for the 59 Basic-safe modules. Full unlocks all 74 modules, localhost and local project scanning, premium modules, and includes PM Privacy Guard. Full Pro adds commercial-use rights for client work plus the same full module set and Privacy Guard bundle.
What do free scans show? ▼
Free scans run the shipped scanner stack as summary-only proof. You can see whether issues were found and how severe the result is. Paid licenses unlock detailed findings, file locations, remediation guidance, exports, and the local dashboard.
Does it run offline? ▼
Yes. PM AIDefense is a self-hosted Node.js tool that runs on your machine. Scans do not require uploading your repo, prompts, configs, or findings. Licenses validate locally with an offline ED25519 signature.
What are the scan profiles? ▼
quick runs the fastest high-signal checks for day-to-day use and CI. osint-light adds lightweight exposure checks. half-deep runs the complete 74-module suite for release checks and deeper audits.
Does PM AIDefense send my code anywhere? ▼
No. PM AIDefense sends no code, prompts, configs, findings, or telemetry to Permanently Mobile. Results stay local in your project output, local database, and browser dashboard.
