PM AIDefense CLI reference

Local CLI commands for AI-agent security scans.

PM AIDefense ships as the npm package @permanentlymobile/pm-aid@1.1.0. The public package contains 74 shipped JS-native modules and runs repository checks locally.

Common commands

npx pm-aid scan --profile quick ./repo
npx pm-aid scan --profile baseline-free ./repo
npx pm-aid scan --profile half-deep ./repo
npx pm-aid scan --list --json
pm-aid license activate <KEY>
