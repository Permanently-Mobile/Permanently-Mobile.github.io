PM AIDefense agent use

Run a local security check before agent-generated code gets merged.

PM AIDefense is a local security scanner for AI-agent repositories. It checks prompts, MCP configs, dependencies, secrets, and runtime files without uploading code or spending LLM tokens. Use it before merging agent-generated code, deploying MCP tools, or running CI checks for AI-agent projects.

Best fit

  • Autonomous coding agents preparing a pull request.
  • Developers checking MCP configs, prompt files, dependencies, and runtime files.
  • Small teams that need local proof without uploading source code.

Copyable first command

npx pm-aid scan --profile quick ./repo

Free mode runs the complete shipped stack as summary-only proof. Paid tiers unlock details, locations, remediation, and reports.

What it checks

Prompt injection, MCP configuration risk, secrets, dependency and supply-chain issues, CI and agent-runtime abuse surfaces.

What it avoids

It does not upload code and does not spend Codex, Claude, OpenAI, or other LLM tokens to inspect a repository.
